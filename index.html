💡 Conseils réservation

  • Réserver AVANT le départ (juillet/août = plein)
  • Toujours demander "piazzola nuda + corrente"
  • Pitchup.com ou camping.info pour comparer
  • Prévoir rallonge 25m (certains emplacements loin)
  • Fossibot F2400 : brancher la nuit, utiliser frigo sur batterie le jour

⚡ Autonomie Fossibot F2400

  • 2400 Wh → frigo 12V ~4-5j autonomie réelle
  • Avec élec camping : recharge complète en ~3h (800W)
  • Panneau solaire recommandé pour les jours sans élec
  • Sans élec 2j max confort avec frigo + éclairage

🚴 Sorties vélo by stop

  • Levanto : Piste littorale Levanto→Bonassola (5 km, tunnel voûté, facile, bluffant)
  • Lucca : Tour des remparts médiévaux (4 km sur les murs, unique en Europe)
  • San Gimignano→Volterra : Route toscane ~40 km, vallonné, vigne & cyprès
  • Val d'Orcia : Strade Bianche (chemins blancs graviers), cadre de carte postale
  • Lago Trasimeno : Tour complet du lac 50 km, quasi plat, superbe

📱 Apps utiles vélo

  • Komoot — traces GPX Italie toutes dispo
  • Strava — segments locaux populaires
  • Wikiloc — traces locales hors-piste

🔧 Emporter dans le Trafic

  • Vélos rack arrière ou galerie toit
  • Pompe + kit crevaison
  • Antivol U (campings = sûrs mais villages touristiques pas toujours)

🛣️ Route — 2 options depuis Morez

  • Option A (rapide) : A40 → Tunnel Mont-Blanc → A5 → A10 Ligurie. ~4h, péages ~35€
  • Option B (col) : D1091 col du Montgenèvre → Briançon → Fréjus → A8 Nice → Vintimille. ~5h, magnifique

🏊 Spots baignade résumé

  • Diano Marina : plage de galets, eau claire ligure
  • Monterosso al Mare : seule vraie plage de sable des Cinque Terre
  • Plage Levanto : sable, moins fréquentée, idéale famille
  • Terme di Saturnia : ⭐ cascades thermales 37°C, GRATUIT, accès libre
  • Lago Trasimeno : eau douce chaude, plages tranquilles, idéal vélo+baignade
  • Marina di Pisa : plage de sable, 20 min de Pise

💶 Budget estimé 15j

  • Campings terrain nu : 15-25€/nuit → ~300€
  • Carburant (diesel ~1.8€/L, ~1500 km) : ~200€
  • Péages : ~60€ aller-retour
  • Nourriture (courses + restos) : ~600€
  • Activités/entrées : ~100€
  • Total estimé : ~1 300€ pour 2

📋 Checklist Trafic

  • Vignette autoroute Italie (pas nécessaire Trafic >3.5T attention)
  • Gilet jaune + triangle dans habitacle (obligatoire IT)
  • Assurance internationale (carte verte)
  • Rallonge électrique 25m + multi-prise
  • Eau : 20L bidon (certains campings comptent l'eau)
  • ZTL Italie : ne jamais entrer en voiture dans les centres historiques !
